Команды


Команды вывода:

	- ls - выводит файлы находящиеся в директории | -l выводит список - ltr - побновлению
	- echo ""  - Выводит текст
	- uptime - показать время 
	- cat - выводит файл в командную строку
	- more - выводит файл в командную строку, контролируя перемещение
	- less - выводит файл в командную строку, контролируя перемещение, может искать слова
	- cut – вывести определенное поле из текста
	- env -окружение 
	- head -n
	- tail -n -f
	- pushd %папка - поместить папку в стек
	- pushd +1 - пройти в папку
	- dirs -v вывести папки в стеке
	- popd - удалить последнюю папку
	
	- uptime - показать время 



Пермещение по папкам:
	- cd - перемещение по папкам




Создание радактирование папок и файлов
	
	- touch - создать файл
	- rmdir - стереть путую дирректориию 
	- cp %файл /путь куда копировать  - Копировать файл
	- cp -R %папка /путь куда копировать  - Копировать файл
	- rm  - удалить файл или директорию
	- mv переименовать файл - файл1 файл2
	- rm  - удалить файл или директорию
	- ln   - создать дубликат файла 
	- grep  - поиск определонного слова в файле            и вывод строк с этим словом | -i отмена регистров
	- ln –s   - создать symbolic линк на файл или директорию типа Shortcut

- man - помощь

- man -k %начало ключа - выводит все команды с %начало ключа

- whereis %команда - выводит где ныходится команда 

- locate %команда - выводит где ныходится команда 




Архивирование
	tar cf  mytar.tar  Folder1   - заархивировать Folder1
	tar xf mytar.tar  - разархивировать архив
	gzip     / bzip2     / xz      – скомпрессировать файл
	gunzip /  bunzip2 / unxz  – раскомпресировать файл

	tar cvzf myBZIP2.bz2  Folder1    – сжать Folder1
	tar xvf  myBZIP2.bz2                  - распаковать архив
	tar tf myBZIP2.bz2    - посмотреть что внутри архива

	zip –r myZIP.zip Folder1   - Запаковать Folder1 в ZIP
	unzip myZIP.zip                - Распаковать файл myZIP.zip
